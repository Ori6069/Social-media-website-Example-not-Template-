# Social-media-website-Example-not-Template-
An easy simple HTML and JS code for a website to publish posts and images. (Just like social media) :)

# How to setup?
1. Get Supabase and create an account if you don't have one.
![Alt](https://github.com/Ori6069/Social-media-website-Example-not-Template-/blob/a6eabb2784a661295bfa84bf9c2cbf0d43944a37/Images/Screenshot%202026-02-14%20182320.png)
2. The URL under your project name is important so keep it, because you will need to put in the code here:
![Alt](https://github.com/Ori6069/Social-media-website-Example-not-Template-/blob/c38a4475ab0f2f4c93c26d23c1d6c6245b057046/Images/Screenshot%202026-02-14%20184427.png)
3. For the ANON key, go to Project Settings.
![Alt](https://github.com/Ori6069/Social-media-website-Example-not-Template-/blob/cc58d21d3707713caf0a7a70ccace39d669e9ab0/Images/Screenshot%202026-02-14%20182346.png)
4. Go to API, then select "Legacy ANON". And thats your ANON key. Place it in your code. Now lets work on the uploading functionality. (Note: name everything excatly as shown if your a beginner.
![Alt](https://github.com/Ori6069/Social-media-website-Example-not-Template-/blob/cc58d21d3707713caf0a7a70ccace39d669e9ab0/Images/Screenshot%202026-02-14%20185606.png)
![Alt](https://github.com/Ori6069/Social-media-website-Example-not-Template-/blob/cc58d21d3707713caf0a7a70ccace39d669e9ab0/Images/Screenshot%202026-02-14%20185621.png)
